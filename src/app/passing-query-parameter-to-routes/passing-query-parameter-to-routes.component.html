Query parameters are optional parameters that we pass to a route. This query parameters are added at the end of the URL seperated by "?"

The difference between route parameter and query parameter is, the routes parameters are required and is used by angular router to determine the route and they are part of rout definition

Query parameter is not a part of rout definition
