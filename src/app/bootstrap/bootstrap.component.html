How to add bootstrap in angular 17

  • In the terminal npm i bootstrap --save
  • Than npm i at-the-rate popperjs/core
  • Go to node_module then bootstrap then dist then css then bootstap.min.css. Copy the relative path and paste it on the angular json file's style section
  • Similarly go to node_module then bootstrap then dist then css then bootstap.min.js. Copy the relative path and paste it on the angular json file's script section
