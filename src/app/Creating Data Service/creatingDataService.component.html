Explanation

Our Targen is to create a service such that it will store the user data and display it

First Creating a service.ts file

Now creata a class and export it and call it UserService

In side this class we created a array of users, inside the array we have list of users (name and status)

Now we want to show the data in the service in our webpage

We are going to assign users property of this UserService class to a property in the parent component

Let create a property in parent class say users which will store array of objects

users:(curly braces)name:string, status:string(curly braces)[]=[];

Now implementing ngOnInit method

Inside this ngOnint we want to assign the users property of this parent component class with the users property of UserService class

Inorder to do that we need the instance of UserService class in our parent component class. So accesing it with help of constructor: constructor(private userServices:UserService)

Now we have the instance of UserService as userservice in parent component class

We have tell angular that we need instance of UserService class but we havn't tell angular how it can provide . So in the component metadata add providers array

Now we can access this UserService in ngOninit:this.users= this.userServices.users

Now in the parent component html file use *ngFor loop to iterate the data and the data will be available in the didplay

Now our aim is to create a form that will recieve user entry: add user

With the help of two way data binding we can do that

In the service.ts flie create a new method say: AddNewUsers

This method will push the data

AddNewUser(name:string, status:string)(curly braces) this.users.push((curly braces)name:name,status:status(curly braces)); (curly braces)

Now we have to call this method in add user component

We need an instance of UserService class in adduser. So creating a constructor: constructor(private userService:UserService)

Now initially assign the property username with empty string and status active

Now binding the username property with to the input element using two way data binding [ngModel]="username"

Similarly for active status

Inorder to activate ngModel we have to import formsmmodule

Now create a method AddUser() inside the methods: this.userService.AddNewUser(this.username, this.status)
