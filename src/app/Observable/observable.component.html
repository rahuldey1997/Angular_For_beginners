We use Observable to perform asynchronous operations and handel asynchronous data

Observable sends the data in chunks. That is it is streaming the data

It is not waiting for all the data to be available and then send

An observable is a function that converts ordinary stream of data into an Observable stream data. You can think of Observable as a wrapper around the ordinary steam of data

The RxJS (Reactive Extension library for JavaScript) is a JavaScript library, that allows us to work with asynchronous data stream

RxJs has two main players

  • Observable: Stream of data
  • Observer: Subscriber

    • An observer is an subscriber of observable

    Note: observable is similar to promish. It will fetch api data.
