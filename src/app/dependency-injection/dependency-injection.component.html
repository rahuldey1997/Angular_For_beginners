Dependency injection is a technique in which a class receives its dependencies from external sources rather than creating them itself

Explanation:

We need to inform angular we need an instance of react service in our react component

For this we need to create a constructor: constructor()

As an argument of the constructor lets create a parameter enrollService and this enroll service is of type ReactService: constructor(private enrollService:ReactService)

At at-the-rate component providers:[ReactService]

Now create a method on export class: OnEnroll()

Now: this.enrollService.OnEnrollClicked(this.title)

So first we create a constructor, inside the constructor we provided a private parameter and we specified its type as ReactService

Then we specified the providers property for the component decorator, which is an array and inside the array we specified the type.

Then we create a method and within the method we are using the instance to call OnEnrollClicked method.
