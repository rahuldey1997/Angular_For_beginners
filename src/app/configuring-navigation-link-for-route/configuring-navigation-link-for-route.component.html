Absolute and Relative route path

Absolute routing

When we specify / before a path it uses the absolute path.

That is, that path get appended to root url that is localhost4200/child, no matter what is the active routed

Here home is absolute path

../child it will go 1 level up.

Relative routing

When we donot specify / before a path it uses the relative path.

That is, that path get appended to parent url that is localhost4200/parent/child

Here about is relative path

./child is considered as relative path not absolute path.
