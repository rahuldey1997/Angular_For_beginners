Pure And Impure Pipes

What is pure pipe

A pure pipe is that pipe which gets called whenever there is pure change on the input value. By default every pipe is a pure pipe.

What is pure change

Following changes are considered as pure change in data:

If the pipe is being used on a primitive type like string, number boolean etc and if the value of the primitive type changes, it is considered as a pure change.

If the pipe is being used on a reference type input and if the reference of that input changes, than the change is pure change.

Eg: percentage pipe is the pure pipe here.

Note:But if the input is the reference type, and only its property value changes and its reference haas not changed, that is not a pure pipe

What is impure pipe

A pipe is impure pipe if it is not pure. An impure pipe gets called for each change detection cycle and it is performed intensive.

We can set a pipe impure by setting the meta data pure=false
