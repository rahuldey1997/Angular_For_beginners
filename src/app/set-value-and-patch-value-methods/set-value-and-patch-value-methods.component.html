setValue() method

The setValue() method is used to update a form FormControl, FormGroup or FormArray value.

To setValue() method, we pass an object to update the value of a FormControl, FormGroup or FormArray. The structure of the object must match the structure of the FormControl, FormGroup or FormArray which we are trying to update.

Any button on form is of submit type. If the type is not given in button it will take type as submit.
