Structural Directive

A structural directives manupulates the DOM by adding or removing element to or from the DOM on which we use it

When we use structural directive we use a *

Why do we use *? What angular do when it find * behind the scene

When we use a * before a directive name. This * tell angular that it is a structural directive. Behind the scene the angular creats a template for this structural directives

Then it wraps the view within the template.

Our aim is to show the notification on clicking the show notice button

Custom Structural Directive

An attribute directive changes the look and behaviour of element on which we use it.

Structural directive manipulates the DOM by adding or removing elements to or from the DOM on which we use it.

When we add or remove an element from the DOM we add or remove it at some places on the DOM.

So in order to create a structural directive we need two things.

What to remove and from where to remove

When we use an attribute directive to an html element in that case in the constructor of that directive we get a reference to that element

We need that reference in case of structural directive as well.

But in case of structural directive along with the element on which we have used the structural directive, we also need the container at which the view is present

This is important notice

This a notice issued for all website users

The div where the structural directive is added is the view

We want this view in the constructor, so that we can manipulate it.

So angular will inject the view in the constructor as its first argument

constructor(private template: TemplateRef,private viewContainer:ViewContainerRef)

Templete is the what to add or remove and viewContainer is the where to add or remove

Now we have to create an input property

Input('appIf') set displayView(condition:boolean)

if condition true then we want to add the view on the DOM else we will remove it

If true then this.where_to_add..createEmbeddedView(this.what_to_add.);

else this.viewContainer.clear()

Now add the selector to container where to add or remove.
